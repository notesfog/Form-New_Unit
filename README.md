# form-new_unit
📦 Pencatatan Barang Datang - TulusTeknoRent

Aplikasi web sederhana berbasis HTML, CSS, dan JavaScript (Vanilla) untuk mencatat, mengelola, dan memonitor barang masuk (khususnya laptop) secara efisien.

Dirancang untuk kebutuhan operasional seperti:

QC barang masuk
Inventory awal sebelum distribusi / rental
Tracking kondisi unit
🚀 Fitur Utama
📝 1. Input Data Barang
Tanggal penerimaan
No resi / pengiriman
Merk & model (dengan katalog otomatis)
Serial number
Jumlah unit
Kondisi:
Baru
Second
Rusak / Servis
Penerima & vendor
Spesifikasi (dropdown + custom input)
Catatan tambahan
Upload multi foto
📊 2. Dashboard Summary

Menampilkan ringkasan real-time:

Total barang
Barang kondisi baru
Barang rusak / servis
Barang second
📋 3. Table Data Interaktif
Menampilkan seluruh data barang masuk
Thumbnail foto
Detail lengkap tiap item
Aksi:
Lihat detail
Edit data
Hapus data
🔍 4. Filter & Search
Pencarian berdasarkan:
Nama
Merk
Serial number
Filter:
Kondisi
Merk
Range tanggal
🖼️ 5. Manajemen Foto
Upload banyak foto sekaligus
Preview langsung
Modal preview (zoom foto)
💾 6. Penyimpanan Data
Menggunakan Local Storage (browser)
Tidak membutuhkan database/server
Data tetap tersimpan selama tidak dihapus manual
📤 7. Export & Print
Export data ke CSV
Print / save sebagai PDF
🔔 8. UX Enhancement
Toast notification (success / error / info)
Modal untuk detail & edit
Responsive layout
Clean UI modern
🧱 Teknologi yang Digunakan
HTML5
CSS3 (Custom Styling, Responsive)
JavaScript (Vanilla)
LocalStorage API
📂 Struktur File
pencatatan-laptop.html

Semua sistem (UI + logic + data handling) berada dalam satu file:

<style> → desain UI
<script> → logic aplikasi
HTML → struktur layout
⚙️ Cara Menggunakan

Download file:

pencatatan-laptop.html
Buka di browser:
Chrome (recommended)
Edge
Firefox
Mulai input data melalui form
Data otomatis tersimpan di browser
⚠️ Catatan Penting
Data tersimpan lokal di browser
Jika:
Clear cache
Ganti browser
Ganti device
→ data akan hilang

👉 Disarankan rutin:

Export CSV sebagai backup

Tidak butuh internet (offline-ready)

Cepat & ringan

IT Support
