# 📦 Pencatatan Barang Datang - TulusTeknoRent

Aplikasi web sederhana berbasis **HTML, CSS, dan JavaScript (Vanilla)** untuk mencatat, mengelola, dan memonitor barang masuk (khususnya laptop) secara efisien.

---

## 🚀 Fitur Utama

### 📝 Input Data Barang
- Tanggal penerimaan
- No resi / pengiriman
- Merk & model (katalog otomatis)
- Serial number
- Jumlah unit
- Kondisi (Baru / Second / Rusak)
- Penerima & vendor
- Spesifikasi (dropdown + custom)
- Catatan tambahan
- Upload multi foto

---

### 📊 Dashboard Summary
- Total barang
- Kondisi baru
- Rusak / servis
- Second

---

### 📋 Table Data
- View, edit, delete data
- Thumbnail foto
- Detail lengkap

---

### 🔍 Filter & Search
- Cari nama, merk, serial
- Filter kondisi, merk, tanggal

---

### 🖼️ Foto
- Multi upload
- Preview & modal zoom

---

### 💾 Penyimpanan
- LocalStorage (tanpa database)

---

### 📤 Export
- CSV
- Print / PDF

---

## 🧱 Teknologi
- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API
